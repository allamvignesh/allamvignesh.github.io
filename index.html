Robotics projects

Projects are presented around my actual contribution, especially where the work was completed as part of a team.

01 Real robot · Team project

Manipulation and execution

Tracy Dual-Arm Manipulation

Planning and integration work for a dual-UR10e platform using Giskard and Coraplex, with real Robotiq gripper control and higher-level manipulation behaviours.

My contribution

Integrated ROS 2 gripper actions into the execution stack, developed alternative real-hardware motion mappings, worked on arm poses and handover behaviour, and connected perception-team outputs to manipulation tasks.

  • ROS 2 action clients for parallel grippers
  • Giskard-compatible execution integration
  • Pick-and-place, stacking, and handover workflows
ROS 2GiskardCoraplex UR10ePython
02 Mobile robotics · Team project

Autonomous navigation

Tortugabot Navigation Planner

A custom navigation stack built without Nav2 for a mobile robot using live occupancy-grid maps and RViz goal input.

My contribution

Implemented A* path planning, map/grid conversions, obstacle inflation, path simplification, waypoint progression, and planning-supervisor logic for communication with the motion controller.

ROS 2A*SLAM Toolbox Occupancy GridRViz2
03 Simulation · University project

Planning and integration

Navigate-to-Grasp Mobile Manipulator

An end-to-end PyBullet project in which a mobile manipulator navigates to a table and grasps a target object.

My contribution

Worked primarily on navigation, obstacle avoidance, base positioning, inverse-kinematics execution, and integration of detected object poses into the grasp sequence.

PyBulletPythonIK NavigationIntegration
